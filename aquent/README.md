# Welcome to my boring company

To get start: 

```
cd crud-app/aquent
yarn install
yarn run dev

```

Open [http://localhost:8080](http://localhost:8080) and lets build our company together


Or...

```
cd crud-app/aquent
yarn install
yarn run build
yarn start

```

Open [http://localhost:3000](http://localhost:3000) to play around the boring company
